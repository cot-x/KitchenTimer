<<説明>>
    「きっちんたいま～」です。

<<動作環境>>
    .Net Framework 1.1 以降の入った環境

<<使用方法説明>>
    起動して、時間を指定して、「すた～と」を押せばカウントダウンが始まります。
    指定時間経過後、メッセージボックスを表示してお知らせします。
    また、表示面を右クリックすることでメニューが表示されます。（終了できます。）

<<color.cfgファイルについて>>
    きっちんたいま～.exeと同じフォルダ内にcolor.cfgファイルを作ると、
    自動的にそれを読み込んで、きっちんたいま～の色が変わります。
    指定方法は、色の名前を書いてください。
    指定できる名前は、
        BLUE
        BROWN
        GRAY
        GREEN
        ORANGE
        PINK
        RED
        WHITE
        YELLOW
    です。
    ファイルが存在しない場合、指定が間違っている場合はWHITEになります。
